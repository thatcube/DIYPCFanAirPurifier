# DIY PC Fan Air Purifier — Project Context

## Project Overview
Interactive 3D model of a DIY air purifier, built as a single HTML file using Three.js r128. The model shows a box-style air purifier constructed from birch plywood, 8× Arctic P12 Pro 120mm fans, and 20×25×1 MERV 13 3M filters. It includes a room scene (bed, nightstand, wall, baseboard) for scale context.

## Key Code Architecture

### Plywood Panels
- **Fan panels** (front/back): `ExtrudeGeometry` with bevel-compensated dimensions
  - `bevelSize: 0.12, bevelThickness: 0.12, bevelSegments: 3`
  - Shape shrunk by `bevelSize` on all edges, holes enlarged by `bevelSize` to compensate
- **Cap panels** (top/bottom): Also bevel-compensated, rotated from XY to XZ plane
- **Hole roundover**: `TorusGeometry(holeR, bR=0.2, 12, 64)` positioned at `zFace*(ply/2 - bR)` — no rotation needed, sits flush at hole edge

### Wood Stain System
3-state toggle via `setStain('raw'|'oil'|'walnut')`:
```javascript
STAIN_COLORS = {
  raw: 0xd2b48c,      // light birch
  oil: 0x8b6340,       // oil stain
  walnut: 0x4a2c17     // dark walnut
}
```
All birch materials tracked in `allBirchMats[]` array.

### Layout System
`setLayout('fb'|'ft')` toggles between two configurations:
- **Front+Back (default)**: Fans on front and back panels, open top
- **Front+Top**: Fans on front and top, solid back panel
- Groups: `backGroup`/`parts.top` (default) vs `altBackGroup`/`altTopGroup` (alternate)
- Top fan rotors: `userData.axis=(0,-1,0)`, `rotation.x=Math.PI/2`

### Fan Wiring
- Bus wire along fan edges (dark gray `0x444444` for visibility in dark scene)
- White/black JST connectors at each fan
- Lead wires to PSU/ESP32 components

### Camera
```javascript
polar = 1.05
azimuth = -0.5 + Math.PI   // flipped 180° to view from +Z side
radius = 65
```

### Room Scene
- **Wall**: at Z=+50, baseboard at Z=+49.5
- **Bed**: Zinus Queen Piper, `bedX = -bedW/2 - 12`, `bedZ = 49.75 - bedL/2` (headboard flush against wall)
- **Nightstand**: `tblZ = 49.75 - tblD/2 - 2` (pulled forward so it's not on top of purifier)

### Material Arrays (for toggling)
- `allFanMats[]` — fan frame materials
- `allFanGlows[]` — fan LED glow meshes
- `allBirchMats[]` — all wood materials (for stain changes)
- `allRotors[]` — fan rotor meshes (for spin animation)
- `allLegMeshes[]` — furniture leg meshes

### Texture
- Wood grain: `TextureLoader` with reduced repeat (`panelW/24, H/24`) for visible grain
- Applied to all birch plywood surfaces

## Build History (Key Fixes)
1. **Black screen** — `bedX` referenced before `const` declaration → moved bed dim declarations above nightstand code
2. **Bed through wall** — wrong Z calc → `bedZ = 49.75 - bedL/2`
3. **Wood too light** — base color `0xffffff` washed out under studio lighting → `0xd2b48c`
4. **Rounded hole edges** — many failed approaches (torus halos, LatheGeometry, cylinders) → torus with no rotation, sunk back by `bR` worked
5. **Rounded outer plywood edges** — cylinder approach ugly → bevel with pre-compensated shapes
6. **Orientation flip** — moved wall/bed to +Z side and rotated camera by `+Math.PI` instead of swapping fan panel params

## UI Controls
- **Fan segment toggles**: Click individual fan buttons to toggle on/off (affects glow, spin, material)
- **Stain selector**: 3 buttons — Raw / Oil / Walnut
- **Layout toggle**: Front+Back / Front+Top
- **Mouse orbit**: Click-drag to rotate, scroll to zoom
