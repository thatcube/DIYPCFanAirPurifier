# DIY PC Fan Air Purifier ā€” Project Context

## Project Overview
Interactive 3D model of a DIY air purifier, built as a single HTML file using Three.js r128. The model shows a box-style air purifier constructed from birch plywood, 8Ć— Arctic P12 Pro 120mm fans, and 20Ć—25Ć—1 MERV 13 3M filters. It includes a full bedroom scene (4 walls, bed, nightstand, TV, window, door) for scale context.

## Key Code Architecture

### Fan Design (P12 Pro style)
- **7 sickle-shaped blades**: sweep=0.52, 14 steps, quadratic sweep (tĀ²), width fills gapAngle*0.82
- **Pinwheel support brace**: 4 spokes, cornerDist=fanFrame*0.44, tangentialOffset=0.35 rad
- **Brace dimensions**: radius=0.07, depth=fanDepth*0.12, positioned at fanDepth*0.34 from center
- **Orientation**: hub cap faces inward, brace faces outward (visible from room)

### Fan Color System
- Constants defined early (~line 208): `FAN_WHITE`, `FAN_BLACK`, `BLADE_FROSTED`
- `fansWhite=true` (default), `fansRGB=true` (default ā€” frosted translucent blades)
- `allFanMats[]` ā€” stride-4 array: [frame, hub, cap, blade] per fan
- `allBraceMats[]` ā€” separate array for brace spoke materials
- Colors applied at init after all fan panels created (~line 729)
- `applyBladeStyle()` ā€” switches between frosted RGB and solid blade modes

### RGB LED System
- `RGB_COLORS`: red (intensity 1.2), yellow/green (1.0)
- Static mode: blade emissive 0.08, cap 0.04, filter 0.03, wood 0.015
- Rainbow/party mode: HSL hue cycling at t*0.3, per-fan phase offset, intensity 1.2
- `allFanGlows[]` ā€” PointLight per fan, range=45
- Toggle: "RGB Fans" toggle switch (on by default), with collapsible LED color section

### Plywood Panels
- **Fan panels** (front/back): `ExtrudeGeometry` with bevel-compensated dimensions
  - `bevelSize: 0.12, bevelThickness: 0.12, bevelSegments: 3`
  - Shape shrunk by `bevelSize` on all edges, holes enlarged by `bevelSize` to compensate
- **Cap panels** (top/bottom): Also bevel-compensated, rotated from XY to XZ plane
- **Hole roundover**: `TorusGeometry(holeR, bR=0.2, 12, 64)` positioned at `zFace*(ply/2 - bR)`

### Particle System
- **600 particles**, two modes: stagnant Brownian drift (fans off) / directional flow (fans on)
- **T breakpoints**: T1=0.40 (intake), T2=0.48, T3=0.62, T4=0.75, T5=1.0
- `pathX(t,side,exhX)` ā€” ease-in intake from W/2+ft+40, transitions to random exhX inside box
- `pathY(t,y0)` ā€” funnels to filter range, clamps to fan stack (fanBot to fanTop) after T2
- `pathZ(t,z0,exitDir)` ā€” funnels intake Z, accelerating exhaust (20 units, ease-out)
- `particleExhX[]` ā€” per-particle random X across 85% panel width for exhaust spread
- `initStagnant(i)` ā€” room-scale random positions (70Ć—(H+20)Ć—60), slow Brownian drift

### Dimension Lines
- `dimTube()` ā€” CylinderGeometry helper for visible line thickness (radius 0.06 main, 0.04 ticks, 0.03 leaders)
- Text sprites at scale 1.8
- Mutual exclusion with exploded view

### Wood Stain System
3-state toggle via `setStain('raw'|'oil'|'walnut')`:
- All birch materials tracked in `allBirchMats[]` array

### Layout System
`setLayout('fb'|'ft')` toggles between two configurations:
- **Front+Back (default)**: Fans on front and back panels, open top
- **Front+Top**: Fans on front and top, solid back panel
- Groups: `backGroup`/`parts.top` (default) vs `altBackGroup`/`altTopGroup` (alternate)

### Room Scene ā€” 4-Wall Bedroom
- **Back wall (Z=50)**: Split into left portion + recessed door alcove (flush to right wall at X=60)
  - Recess: X=24 to X=60, depth=18" inward (recessZ=32)
  - Left return wall connects back wall to recessed section
  - Six-panel door (32"W Ć— 80"H) centered in recess, room-facing
  - Door frame trim, brushed nickel knob + base plate
  - All pieces collected in `backWallParts[]` for fade
- **Front wall (Z=-50)**: Solid wall with 65" OLED TV (centered on bed)
- **Right wall (X=60)**: Solid wall with baseboard
- **Left wall (X=-60)**: Window wall (4 pieces around opening) with curtains + rod
  - Window centered on bed Z position (`winCenterZ=bedZ`)
  - Key light positioned from left wall (window direction)
- **Floor**: Carpet at floorY=-10
- **Bed**: Zinus Queen Piper (82.3"L Ć— 60.3"W Ć— 42"H), `bedX=-bedW/2-12`, headboard against back wall
- **Nightstand**: Between bed and purifier, with books and coffee mug

### Wall Auto-Fade
- Proximity-based: walls are fully opaque at distance, fade to `fadeMin=0.06` within 25 units of wall plane
- All 4 walls + door alcove parts fade independently based on camera position
- Back wall fade also applies to door panel

### UI
- Frosted glass panels with `backdrop-filter: blur(24px) saturate(1.4)`
- Toggle switches for: exploded view, show filters, dimensions, X-ray, spin fans, show airflow, RGB fans
- Segment buttons for: camera angle, fan color, LED color, wood stain, feet style
- RGB section uses `grid-template-rows` animation for smooth collapse/expand
- Liquid button animations: spring cubic-bezier on hover/active, toggle switch knob stretch on press
- FAB button to show/hide panel
- Day/night mode toggle
